**External URL risk:**
- `DungeonsAndDragons/index.html:205-213` — 9 stone tile images loaded via `https://raw.githubusercontent.com/amandarae220/DungeonsAndDragons/main/stone_X.png`. These files exist in the repo and the URLs are live, but GitHub raw CDN imposes rate limits and these URLs will break if the repo is renamed, made private, or the branch changes. Fix: Serve images as relative paths from the same repo.
